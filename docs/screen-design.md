## 4. コンポーネント一覧（Next.js）

| コンポーネント名 | 用途 |
|---------------|------|
| Header | 全画面共通ヘッダー（タイトル・ナビゲーション） |
| RecipeCard | レシピ一覧のカード1枚 |
| RecipeGrid | RecipeCard の一覧表示グリッド |
| SearchBar | 検索入力フォーム |
| FilterBar | ジャンル・調理時間・タグのフィルター群 |
| IngredientList | 材料一覧（チェックボックス付き） |
| InstructionList | 手順一覧（ステップ番号付き） |
| NutritionCard | 栄養情報の表示カード |
| RecipeForm | レシピ登録・編集フォーム（P03/P04 共用） |
| ImageUploader | 画像ドラッグ&ドロップ + プレビュー |
| IngredientInput | 材料行の入力フォーム（動的追加対応） |
| InstructionInput | 手順行の入力フォーム（動的追加対応） |
| ShoppingList | 買い物リスト全体 |
| ShoppingItem | 買い物アイテム1行（チェック・削除） |
| Modal | 削除確認・人数入力などの汎用モーダル |
| Pagination | ページネーション or 無限スクロール |

## 5. レスポンシブ対応方針

| ブレークポイント | レシピグリッド | その他 |
|--------------|------------|------|
| ～ 639px（スマートフォン） | 1カラム | ハンバーガーメニュー |
| 640px ～ 1023px（タブレット） | 2カラム | 通常ナビ |
| 1024px 〜（PC） | 3カラム | 通常ナビ |

- タップターゲット最小 44×44px
- 調理中のスマートフォン操作を想定し、重要ボタンは画面下部に固定（FAB等）
